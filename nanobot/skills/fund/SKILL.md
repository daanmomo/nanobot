# 基金分析技能

你具备分析中国公募基金的能力，可以帮助用户：

## 功能概述

### 1. 净值查询
- 获取基金最新净值和涨跌幅
- 获取历史净值走势

### 2. 持仓分析
- 查看基金前十大重仓股
- 了解基金投资方向

### 3. 基金排名
- 按收益率排名（近1周/1月/3月/1年/3年）
- 按类型筛选（股票型/混合型/债券型/指数型/QDII）

### 4. 基金搜索
- 通过代码或名称搜索基金

## 可用工具

- `fund_nav` - 获取基金净值
- `fund_nav_history` - 获取历史净值
- `fund_holdings` - 获取基金持仓
- `fund_ranking` - 获取基金排名
- `fund_search` - 搜索基金
- `fund_info` - 获取基金信息

## 使用示例

### 查询净值
```
用户: 查看易方达蓝筹的净值
助手: [调用 fund_search，keyword="易方达蓝筹"] 然后 [调用 fund_nav]
```

### 查看持仓
```
用户: 005827 这只基金持仓是什么？
助手: [调用 fund_holdings，fund_code="005827"]
```

### 基金排名
```
用户: 今年收益最好的股票型基金有哪些？
助手: [调用 fund_ranking，fund_type="股票型"，sort_by="近1年"]
```

## 注意事项

1. **数据来源**: 数据来自天天基金网（东方财富）
2. **净值更新**: 基金净值通常在交易日晚间更新
3. **持仓数据**: 持仓数据为季报披露数据，有一定滞后
4. **风险提示**: 数据仅供参考，不构成投资建议

## 常见基金类型

| 类型 | 特点 | 风险等级 |
|------|------|---------|
| 股票型 | 股票仓位≥80% | 高 |
| 混合型 | 股票+债券灵活配置 | 中高 |
| 债券型 | 主要投资债券 | 中低 |
| 指数型 | 跟踪特定指数 | 中高 |
| 货币型 | 投资货币市场工具 | 低 |
| QDII | 投资海外市场 | 高 |
