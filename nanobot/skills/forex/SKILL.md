# 外汇分析技能

你具备查询和分析外汇汇率的能力，可以帮助用户：

## 功能概述

### 1. 实时汇率
- 获取指定货币对的实时汇率
- 获取主要货币对人民币汇率

### 2. 汇率换算
- 支持主要货币之间的换算
- 自动获取最新汇率计算

### 3. 历史汇率
- 查看货币对的历史汇率走势

## 可用工具

- `forex_rate` - 获取实时汇率
- `forex_major_rates` - 获取主要汇率
- `forex_convert` - 汇率换算
- `forex_history` - 获取历史汇率

## 使用示例

### 查询汇率
```
用户: 今天美元汇率多少？
助手: [调用 forex_rate，currency_pair="USD/CNY"]
```

### 查看主要汇率
```
用户: 人民币对主要货币的汇率
助手: [调用 forex_major_rates]
```

### 汇率换算
```
用户: 1000美元能换多少人民币？
助手: [调用 forex_convert，amount=1000，from_currency="USD"，to_currency="CNY"]
```

### 历史汇率
```
用户: 最近一个月美元汇率走势
助手: [调用 forex_history，currency_pair="USD/CNY"，days=30]
```

## 注意事项

1. **数据来源**: 汇率数据来自中国银行
2. **汇率类型**: 提供的是银行中间价/买入价
3. **更新频率**: 银行汇率每个工作日更新
4. **换算说明**: 实际兑换时银行会收取点差

## 货币对格式

货币对格式为 `基础货币/报价货币`，例如：
- `USD/CNY` - 美元/人民币（1美元 = X人民币）
- `EUR/CNY` - 欧元/人民币
- `GBP/CNY` - 英镑/人民币
