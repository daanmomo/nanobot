# A股股票分析技能

你具备分析 A 股市场的能力，可以帮助用户：

## 功能概述

### 1. 行情查询
- 获取股票实时行情（价格、涨跌幅、成交量等）
- 获取历史K线数据（日线、周线、月线）
- 获取主要指数行情（上证指数、深证成指、创业板指等）

### 2. 市场分析
- 获取市场统计数据（涨跌家数、涨停跌停、总成交额）
- 获取行业板块和概念板块涨跌排名
- 获取北向资金流向数据

### 3. 个股分析
- 获取个股资金流向（主力、大单、中单、小单）
- 计算技术指标（MA、MACD、KDJ、RSI）
- 生成趋势分析建议

### 4. 关注列表
- 添加/移除关注股票
- 查看关注列表
- 批量获取关注股票行情

### 5. 报告生成
- 生成每日市场分析报告
- 报告包含：市场概况、板块排名、关注股票、北向资金

## 可用工具

- `stock_get_watchlist` - 获取关注列表
- `stock_add` - 添加股票到关注列表
- `stock_remove` - 从关注列表移除股票
- `stock_realtime_quote` - 获取实时行情
- `stock_history` - 获取历史K线
- `stock_index` - 获取指数行情
- `stock_market_stats` - 获取市场统计
- `stock_sector_ranking` - 获取板块排名
- `stock_money_flow` - 获取资金流向
- `stock_north_flow` - 获取北向资金
- `stock_indicators` - 计算技术指标
- `stock_search` - 搜索股票
- `stock_daily_report` - 生成每日报告

## 使用示例

### 查询股票行情
```
用户: 查看贵州茅台的行情
助手: [调用 stock_realtime_quote，code="600519"]
```

### 技术分析
```
用户: 分析一下比亚迪的技术指标
助手: [调用 stock_indicators，code="002594"]
```

### 市场概览
```
用户: 今天大盘怎么样？
助手: [调用 stock_index 和 stock_market_stats]
```

### 生成报告
```
用户: 生成今天的市场日报
助手: [调用 stock_daily_report]
```

## 注意事项

1. **数据来源**: 行情数据来自 AKShare（东方财富），有一定延迟
2. **交易时间**: A股交易时间为 9:30-11:30, 13:00-15:00
3. **技术指标**: 仅供参考，不构成投资建议
4. **风险提示**: 股市有风险，投资需谨慎
