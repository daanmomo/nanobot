# 美股/港股分析技能

你具备分析美股和港股市场的能力，可以帮助用户：

## 功能概述

### 1. 实时行情
- 获取美股/港股实时行情（价格、涨跌幅、成交量等）
- 获取主要市场指数（标普500、道琼斯、纳斯达克、恒生指数等）

### 2. 历史数据
- 获取历史K线数据（日线、周线、月线）
- 支持多种时间范围（1天到最大历史）

### 3. 公司分析
- 获取公司基本信息（行业、简介、员工数等）
- 获取财务数据（估值、盈利能力、收入、股息等）

### 4. 基本面指标
- 市盈率 (P/E)、市净率 (P/B)、市销率 (P/S)
- 利润率、ROE、ROA
- 收入增长、盈利增长
- 股息率、派息比率

## 可用工具

- `usstock_quote` - 获取实时行情
- `usstock_history` - 获取历史K线
- `usstock_company` - 获取公司信息
- `usstock_financials` - 获取财务数据
- `usstock_indices` - 获取主要指数
- `usstock_search` - 搜索股票

## 使用示例

### 查询股票行情
```
用户: 查看苹果公司的股价
助手: [调用 usstock_quote，symbol="AAPL"]
```

### 公司分析
```
用户: 分析一下微软的财务状况
助手: [调用 usstock_financials，symbol="MSFT"]
```

### 港股查询
```
用户: 腾讯股价多少？
助手: [调用 usstock_quote，symbol="0700.HK"]
```

### 全球指数
```
用户: 美股三大指数今天怎么样？
助手: [调用 usstock_indices]
```

## 注意事项

1. **数据来源**: 行情数据来自 Yahoo Finance，有一定延迟
2. **交易时间**: 美股交易时间为美东时间 9:30-16:00
3. **港股代码**: 港股代码需要加 .HK 后缀（如 0700.HK）
4. **风险提示**: 数据仅供参考，不构成投资建议
