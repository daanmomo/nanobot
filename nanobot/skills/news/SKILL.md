# 财经新闻技能

你具备获取和分析财经新闻资讯的能力，可以帮助用户：

## 功能概述

### 1. 财经要闻
- 获取最新财经新闻头条
- 了解市场热点动态

### 2. 股票新闻
- 获取个股相关新闻
- 追踪公司动态

### 3. 公司公告
- 获取上市公司公告
- 了解重要信息披露

### 4. 研报资讯
- 获取券商研究报告
- 查看机构评级

### 5. 财经快讯
- 实时财经快讯
- 第一时间获取市场信息

## 可用工具

- `news_financial` - 获取财经要闻
- `news_stock` - 获取股票新闻
- `news_announcements` - 获取公司公告
- `news_research` - 获取研究报告
- `news_flash` - 获取财经快讯

## 使用示例

### 查看财经要闻
```
用户: 今天有什么重要财经新闻？
助手: [调用 news_financial]
```

### 查看股票新闻
```
用户: 茅台最近有什么新闻？
助手: [调用 news_stock，stock_code="600519"]
```

### 查看公司公告
```
用户: 比亚迪最新公告
助手: [调用 news_announcements，stock_code="002594"]
```

### 查看研报
```
用户: 最新的券商研报有哪些？
助手: [调用 news_research]
```

### 财经快讯
```
用户: 最新的财经快讯
助手: [调用 news_flash]
```

## 注意事项

1. **数据来源**: 新闻数据来自东方财富、新浪财经等
2. **更新频率**: 实时更新
3. **信息性质**: 新闻仅供参考，需自行判断
4. **投资提示**: 不构成投资建议

## 新闻类型说明

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 财经要闻 | 宏观经济、政策、市场要闻 | 了解大盘环境 |
| 股票新闻 | 个股相关报道 | 跟踪持仓股票 |
| 公司公告 | 官方信息披露 | 重要事件跟踪 |
| 研究报告 | 券商分析报告 | 专业分析参考 |
| 财经快讯 | 实时简讯 | 即时信息获取 |
